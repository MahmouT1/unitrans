===============================================
نماذج ملفات .env الصحيحة
===============================================

هذه نماذج لملفات البيئة المطلوبة.
قم بنسخها وتعديل القيم حسب إعدادات السيرفر الخاص بك.

===============================================
📄 backend-new/.env
===============================================

# MongoDB Configuration
MONGODB_URI=mongodb://localhost:27017
MONGODB_DB_NAME=student_portal

# Server Configuration
PORT=3001
NODE_ENV=production

# CORS - Frontend URL
FRONTEND_URL=http://localhost:3000

# JWT Secret (غير هذا إلى قيمة عشوائية آمنة)
JWT_SECRET=your-secret-key-change-this-to-random-string

# Optional: If using MongoDB Atlas
# MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/

===============================================
📄 frontend-new/.env.local
===============================================

# Backend API URL
BACKEND_URL=http://localhost:3001

# MongoDB Configuration (for Next.js API routes)
MONGODB_URI=mongodb://localhost:27017
MONGODB_DB_NAME=student_portal

# Next.js Configuration
NEXT_PUBLIC_API_URL=http://localhost:3000

# Optional: For production with domain
# BACKEND_URL=https://api.yourdomain.com
# NEXT_PUBLIC_API_URL=https://yourdomain.com

===============================================
📝 ملاحظات مهمة:
===============================================

1. اسم قاعدة البيانات:
   - MONGODB_DB_NAME=student_portal
   - يجب أن يكون نفس الاسم في Frontend و Backend

2. JWT_SECRET:
   - غير القيمة الافتراضية إلى نص عشوائي طويل
   - مثال لتوليد قيمة عشوائية:
     node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"

3. المنافذ (Ports):
   - Backend: 3001
   - Frontend: 3000
   - MongoDB: 27017

4. إذا كان المشروع على دومين:
   - Backend: https://api.yourdomain.com
   - Frontend: https://yourdomain.com
   - عدل CORS settings

===============================================
🔧 كيفية إنشاء الملفات على السيرفر:
===============================================

# Backend .env
cd /var/www/unitrans/backend-new
nano .env
# (الصق المحتوى من الأعلى، عدل القيم، احفظ بـ Ctrl+O ثم Ctrl+X)

# Frontend .env.local
cd /var/www/unitrans/frontend-new
nano .env.local
# (الصق المحتوى من الأعلى، عدل القيم، احفظ بـ Ctrl+O ثم Ctrl+X)

===============================================
✅ التحقق من الملفات:
===============================================

# تحقق من وجود الملفات:
ls -la /var/www/unitrans/backend-new/.env
ls -la /var/www/unitrans/frontend-new/.env.local

# اعرض محتوى الملفات (للتأكد):
cat /var/www/unitrans/backend-new/.env
cat /var/www/unitrans/frontend-new/.env.local
