==================================================
🔍 الخطوة 1: شغل هذا Script
==================================================

على السيرفر:

-----------------
1. اسحب التحديثات
-----------------
cd /var/www/unitrans
git pull origin main

-----------------
2. أعط صلاحيات
-----------------
chmod +x show-students-collection.sh

-----------------
3. شغل البحث
-----------------

==================================================
📤 ماذا سيفعل هذا Script؟
==================================================

سيبحث ويعرض:
✅ جميع الجداول في قاعدة البيانات
✅ عدد الطلاب في كل جدول
✅ بنية البيانات الكاملة (جميع الحقول)
✅ نماذج من 5 طلاب حقيقيين
✅ إحصائيات تفصيلية

==================================================
📊 ما أحتاجه منك:
==================================================

بعد تشغيل الـ Script:

1. انسخ كامل النتيجة
2. أرسلها لي
3. سأحلل البيانات وأعرف:
   - اسم الجدول الصحيح (students أو users أو غيره)
   - أسماء الحقول الدقيقة
   - بنية البيانات الفعلية

4. بناءً على ذلك، سأنشئ API route يتوافق 100% مع بياناتك

==================================================
⚠️ ملاحظة مهمة:
==================================================

✅ آمن تماماً - فقط يقرأ البيانات
✅ لن يغير أي شيء
✅ لن يحذف أي بيانات
✅ سريع - 5-10 ثوان فقط

==================================================
✅ جاهز؟ شغل الـ Script الآن!

cd /var/www/unitrans
